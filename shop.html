How can I get the refund for my cancelled order?

A refund will be automatically initiated when your order is:
  • Order is cancelled by seller
  • Refund request is accepted by seller
  • Refund request is accepted by Shopee (in the event of a dispute)
  • Order is automatically cancelled by system due to seller failing to ship order(s)

How do I return my wrong/damaged item once the return request has been confirmed?

Local orders (purchase made with local seller):
  • You will have 3 days to return your request once it is approved by the seller.
  • A return shipping label will be sent to your email address. If you create your own shipping label.
Overseas / cross-border orders:
  • If return is accepted by Dispute team, the return address will be provided by via email from dispute@support.shopee.sg or push notification in Shopee app.
  • Do ensure that you ship out the return item(s) within 3 days from the date your return request was approved.
  • After shipping, remember to enter the Courier Name, Tracking Number (if applicable) or Shipment proof via the Shopee app!

How do I raise a dispute on the Shipping Fee charge?

If you have encountered with overcharged shipping fee from NinjaVan or Speedpost Economy, please email the supporting details as per below to Shopee Customer Support Team or via the Live Chat.
  1. Sample picture of the item & the packaging or after it has been packed
  2. Dimensions (Length x Width x Height) of the item after it has been packed together with the packaging
  3. Sample picture of the dimensions (Please provide picture of the parcel being measured with a tape)
  4. Weight of the item after it has been packed (best if there's a picture of the item on the weighing scale)

How long would it take for my refund to be processed?

All request for refunds or order cancellation that has been successful will be able to see the refund process initiated within 24 hours.
The refund process will be different according to the payment method used to pay for the order.
If you pay using:

1) Credit / Debit Card:

Payment will be released back to the card you used to make payment. This process will take 7-14 working days depending on the card issuing bank.

2) All other payment method: (for orders paid using online banking, ATM/Cash Deposit, ShopeePay)

Payment will be released to your ShopeePay. You will be able to use it to make future purchases through "ShopeePay" or have it withdrawn after you have updated your bank account details.

What is Lazada's Return Policy?

LazMall items can be returned within 15 days upon receipt of shipment.*

Non-LazMall items can be returned within 7 days upon receipt of shipment, and change of mind returns are not allowed. Change of Mind is also know as the return reason "Don't want item anymore" at the returns page.

*If you have changed your mind and would like to return your item, please note that we will only accept returns of LazMall items that:
  • Have not been opened
  • Have all seals and tags intact (if any)
  • Is unworn, unwashed or unaltered (if applicable)
However, change of mind returns are not applicable for the following products:
  • Digital goods, such as mobile top up cards and food vouchers
  • All groceries
  • All paint and primers
  • All undergarments
  • All products that require installation, customisation or are bulky

What is the return policy for bundled items?

For bundled items, all items in the bundle must be returned in order for the return to be processed.

What is the return policy for items delivered by Sellers' own fleet?

You may only return the item if it is defective or damaged. Once the proof of delivery has been signed, you may not return the item under Change of Mind.

What is the time frame in which a return request can be made?

You can make a return request within 15 days (for LazMall items) and 7 days (for Non-LazMall items) from the date that the items were received. Please visit our Returns Page for a list of valid returns reasons.
